## How To Use

In any new Sanctuary repo:

1. Copy `.gitignore-template` → `.gitignore`
2. Copy `CLAUDE-TEMPLATE.md` → `CLAUDE.md` (edit for project)
3. Copy `CHECKLIST-TEMPLATE.md` → `docs/CHECKLIST.md`
4. Reference this repo in your README
